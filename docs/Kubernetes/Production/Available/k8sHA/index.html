    控制面 高可用 #

    1. 控制平面组件划分单独节点；
    2. 控制平面所在节点，应该确保在不同机架上；
    3. 保证控制平面的每个组件有足够的CPU、内存和磁盘资源；
    4. 减少或消除外部依赖； Cloud Provider API
    5. 核心组件以普通Pod形式加载运行时， 可能会调度到任意工作节点。
