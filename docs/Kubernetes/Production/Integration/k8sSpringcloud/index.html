    • 架构

      • k8s ingress 结合 zuul

    • 网关 候选

      • zuul 需要使用 Ingress中配置路由很麻烦

    • 服务配置 候选

      • 环境变量
      • Service mysql-svc
      • ConfigMap
      • apollo

    • 服务注册

      • 老项目 改动Eureka会有风险，建议保留 Eureka
      • 新项目 [4] 用istio envoy side 做 服务发现
      • 候选
        • Service 基于CoreDNS的服务发现
        • Env 基于环境变量的服务发现
